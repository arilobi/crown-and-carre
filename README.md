# E-commerce - Crown and Carre
## By Marion Nabulobi

Crown and Carre is a full-stack e-commerce application designed to enable users to browse and purchase art prints. It pairs a smooth, responsive interface with a secure and efficient backend, creating a seamless shopping experience from end to end.

## Overview
The frontend is crafted with React, using the Context API to manage shared state across the application. On the backend, Flask, Python, and SQLAlchemy work together to handle data persistence, authentication, and all server-side operations.

The interface gracefully adapts to any screen size — desktop, tablet, or mobile — ensuring that users can use the web application comfortably wherever they are.

## Features
It only has two roles: Admin and Client. 

The **Client** can:
- View all the products.
- Write, edit and delete their reviews in real time.
- Can view other reviews from other users.
- Can create, edit and delete their user details.
- Register an account and log in using JWT Authentication for security and privacy.
- View their orders and delete them.
- Can put a product in their wishlist and remove them.

The **Admin** can:
- Add a product, edit and delete.
- View all orders.
- View all products.
- View all users and delete.
- Edit and delete their user details.

## Setup Instructions
1. Clone the repository to your VS Code.
2. Navigate to `cd frontend`
3. Run the command, `npm install` in your VS Code terminal.
4. Run the final command to start the frontend with `npm run build`
